# 介绍
 
**BedrockPlayerSupport** 是一款 Bukkit/Spigot 插件，旨在提升 Minecraft Java 版服务器中基岩版玩家的体验。它为各种服务器命令和功能创建了基于 GUI 的界面，让习惯于触控和表单界面的基岩版玩家能够更加直观地进行交互。
