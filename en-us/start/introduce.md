# Introduce
 
**BedrockPlayerSupport** is a Bukkit/Spigot plugin designed to enhance the experience of Bedrock edition players on Java edition Minecraft servers. It creates GUI-based interfaces for various server commands and features, making interaction more intuitive for Bedrock players who are accustomed to touch controls and form-based interfaces.
